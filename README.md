netty-android
=============

Introduction
 
이 라이브러리는 Netty(http://netty.io/) 프로젝트 v3.8을 Android 용으로 경량화한 버전입니다. 
별도의 소스코드 추가없이 Netty 내부의 코드에서 Android에 불필요한 내용을 삭제하고 정리했습니다.
